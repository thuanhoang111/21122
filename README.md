## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```

## Run Locally

Clone the project

```bash
  git clone https://.......
```

Go to the project directory

```bash
  cd Src_wacaApp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
